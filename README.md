# 비올까

강수 레이더를 백그라운드에서 주기적으로 확인해서, 비가 오기 전에 미리 알려주는 안드로이드 앱.

## 주요 기능

- **강수 레이더 지도**: 앱을 열면 실시간 강수 레이더 지도를 바로 볼 수 있어요.
- **백그라운드 상시 감시**: 앱을 꺼도 배터리 최적화 예외를 받은 포그라운드 서비스가 계속
  주변 강수 레이더를 확인해요.
- **강수 도달 예측 알림**: 위치 기반으로 비구름의 이동 속도와 방향을 분석해서
  "N분 뒤 비가 옵니다", "지금 비가 오고 있어요", "비가 그쳤어요" 알림을 보내줘요.
- **비구름 이동 경로 시각화**: 계산된 비구름 이동 경로를 지도 위에 화살표로 표시해요.

## 기술 스택

- Kotlin, Jetpack Compose
- WebView (강수 레이더 지도 렌더링)
- FusedLocationProviderClient (위치 기반 예측)
- Foreground Service + 배터리 최적화 예외 (백그라운드 상시 감시)
- minSdk 31 / targetSdk·compileSdk 36

## 빌드 & 실행

```bash
./gradlew assembleDebug
adb install -r app/build/outputs/apk/debug/app-debug.apk
```

최초 실행 시 알림 → 위치 → 백그라운드 위치 → 배터리 최적화 제외 순서로 권한을 요청합니다.
모두 허용하면 강수 레이더 지도로 넘어가고, 백그라운드 감시가 함께 시작됩니다.
